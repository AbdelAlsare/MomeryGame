Bienvenue sur le site du Memory de L'ENI.
Venez Jouer avec nous et faites le meilleur score pssible.

    Régle du jeu :

  • Retournez deux cartes.Si les images sont identiques, le jeu laisse les cartes visbles.
  • Si les images sont différentes , les cartes se remoutent facces cachées là où elles étaient.
  • La partie est termineé lorsque toutes les cartesont été assemblées par paires.

Si vous voulez souvenir de votre score. Inscrivrez-vous via le formulare le jeu le 5 meelleurs scores.
Si vous voulez relance une partie cliquez sur la barre d'espace.
